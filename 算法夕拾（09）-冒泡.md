### 原理
两两比对，调整顺序，比较n-1次

### 实现

```
public static void sort(int[] arr) {
	for (int i = 1; i < arr.length; i++) {
		for (int j = 0; j < arr.length - i; j++) {
			if (arr[j] < arr[j + 1]) {
				int tmp = arr[j];
				arr[j] = arr[j + 1];
				arr[j + 1] = tmp;
			}
		}
	}

	for (int i = 0; i < arr.length; i++) {
		System.out.println(arr[i]);
	}
}
```
### 复杂度
时间：O(n2)

空间：O(1)
